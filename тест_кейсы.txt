Приложение "Мой МТС"
1. Тест функции добавления абонента
2. Телефон iPhone 5S, доступ в интернет
3. Шаги теста:
	1. Открыть приложение "Мой МТС"
	2. Нажать кнопку "Добавить абонента"
	3. Ввести номер телефона абонента
	4. Нажать кнопку "Получить пароль по SMS"
	5. Ввести полученный код
	6. Нажать кнопку "Войти"
4. Новый абонент появился в графе абонентов



Приложение "Mi Fit"
1. Тест на полноту перевода приложения
2. Телефон iPhone 5S, браслет Mi Band
3. Шаги воспроизведения :
	1. Открыть переведенное на английский приложение "Mi Band"
	2. Включить bluetooth, подключить браслет
	3. Нажать в область экрана, в которой отображается кол-во шагов
4. Надпись "Today" отобразилась на английском языке



Приложение "Люксор" 
1. Тест функции оформления заказа
2. Телефон iPhone 5S, доступ в интернет
3. Шаги воспроизведения:
	1. Открыть приложение "Люксор"
	2. Выбрать любой фильм из любой категории
	3. Нажать кнопку "Сеансы"
	4. Выбрать любое время
4. Показана схема зала с возможностью оформления заказа
